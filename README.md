# BJF INJECTION CODETOOLS

## 📱 О проекте

**BJF INJECTION CODETOOLS** — это многофункциональный инструмент, разработанный **Jason Smith** и **Zakhaev Nikolay Alelsandr** **Sheik Bin Laden** для анализа файлов, обнаружения вредоносного ПО и реверс-инжиниринга. Доступен на нескольких языках программирования.

## 🚀 Функции

| # | Функция | Описание |
|---|--------|-----------|
| 1 | ДАННЫЕ АРХИТЕКТУРЫ | Анализирует архитектуру файлов/папок |
| 2 | ПОЛУЧИТЬ ДАННЫЕ | Извлекает подробную информацию из APK |
| 3 | ФАЙЛЫ В ПАПКЕ | Показывает все файлы (включая скрытые) |
| 4 | ДОСТУП К КОДУ | Просматривает исходный код файлов |
| 5 | ОБНАРУЖИТЬ ВИРУС | Сканирует файлы на наличие угроз |
| 6 | АНАЛИЗ APK | Полный анализ APK файлов |

## 📦 Доступные версии

- Python (Основная)
- JavaScript / Node.js
- Lua
- Kotlin
- Objective-C

## 🔧 Установка

### Python

```bash
git clone https://github.com/jasonsmith/BJF-INJECTION-CODETOOLS.git
cd BJF-INJECTION-CODETOOLS
pip install -r requirements.txt
python bjf_main.py
```
